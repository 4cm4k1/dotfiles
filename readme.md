## Installation

1.  Clone this repo and `cd` in.

    ```shell
    # For HTTPS
    git clone https://github.com/4cm4k1/dotfiles.git && cd dotfiles
    # For SSH
    git clone git@github.com:4cm4k1/dotfiles.git && cd dotfiles
    ```

2.  Install!

    ```shell
    ./dotfiles/install
    ```
